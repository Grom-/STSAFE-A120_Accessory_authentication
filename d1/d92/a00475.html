Detailed Description

The STSAFE Middleware Platform Abstraction Layer gather all functions used for adapting the Platform specific Hardware and Software .
The function declared in the middleware platform abstraction layer must be adaped by the system developer to fit with target application HW/SW specificty . Please refer to PAL function description to get more details on expected function behavior and recommended implementation
