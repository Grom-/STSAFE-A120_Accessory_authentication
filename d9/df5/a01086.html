STSAFE-A120_Symmetric_key_management_AES-256_CCM

This example illustrates a symmetric key (AES-256 CCM) provisionning and usage in STSAFE-A1xx Secure Element. It uses the STSAFE Middleware APIs to a write a key and encrypt then decrypt a message.
