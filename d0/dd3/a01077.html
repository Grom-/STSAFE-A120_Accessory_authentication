STSAFE-A120_Host_key_provisioning_wrapped

This example illustrates how to performs host key provisioning wrapped of a target STSAFE-A120.

The example applicative flowchart is illustrated below :

STSAFE-A120_Host_key_provisioning_wrapped Example flowchart

API and services used in the example :

  • stse_init
  • stse_host_key_provisioning_wrapped
  • stsafea_query_host_key_provisioning_ctrl_fields
  • stsafea_put_host_key_provisioning_ctrl_fields

When executed following logs can be seen on the host attached computer terminal window
