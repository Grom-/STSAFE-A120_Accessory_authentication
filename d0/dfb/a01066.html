STSAFE-A110_Multi-step_device_authentication

This example illustrates STSAFE-A110 device authentication process using Multi-Step approach . it can be taken as reference for building distant server authentication use cases.

The example applicative flowchart is illustrated below :

STSAFE-A110_Device_authentication Example flowchart

API used in the example :

  • stse_init
  • stse_certificate_parse
  • stse_certificate_print_parsed_cert
  • stse_certificate_get_key_type
  • stse_certificate_verify_signature
  • stse_certificate_is_parent
  • stse_ecc_generate_signature
