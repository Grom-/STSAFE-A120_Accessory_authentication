Detailed Description

The Application Programming Interface (API) layer is the entry point for the upper system application layer.
It provides high level functions to the application layer for secure element management.

[out]pOutPointer to received data buffer
[in]sizeSize in bytes of pIn buffer
Returns
stse_ReturnCode_t : STSE_OK on success ; error code otherwise

Description Following diagram illustrates the interactions performed between the Host and the target STSAFE-A1xx device during the API execution



Use-case example
The following applicative code snippet illustrates how to use this API function in main application.

int main (void)
{
stsafe_ReturnCode_t stsafe_ret;
PLAT_UI16 echo_length = 4;
PLAT_UI8 echo_send_buffer[echo_length] = {0x00, 0x11, 0x22, 0x33};
PLAT_UI8 echo_recv_buffer[echo_length];
stsafe_allocate_handler(stsafe_handler);
stsafe_handler.io.BusType = STSAFE_BUS_TYPE_I2C;
stsafe_handler.io.Busaddr = 3;
stsafe_handler.io.Devaddr = 0x20;
stsafe_ret = stsafe_init(&stsafe_handler);
stsafe_ret = stsafe_se_echo(&stsafe_handler,
echo_send_buffer,
echo_recv_buffer,
echo_length);
}
