Verify the signature in arguments.

This API use the STSE to verify a signature

Parameters
[in]pSTSEPointer to STSE Handler
[in]key_typeSignature key type
[in]pPublic_keyPublic key to verify the signature
[in]pSignatureSignature buffer
[in]pMessageMessage used in signature
[in]message_lengthMessage length
[in]message_is_hashedFlag indicating a hashed message (1) or a plaintext (0)
[out]pSignature_validitySignature validity flag (1 = valid signature, invalid otherwise)
Returns
stse_ReturnCode_t : STSE_OK on success ; error code otherwise
